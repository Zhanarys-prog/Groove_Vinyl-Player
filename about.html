About Groove

A love letter to vinyl — built with code, played in your browser.

Real Vinyl Feel

A fully animated turntable with a tonearm, spinning disc, and needle drop — all built in pure CSS and JS. No canvas, no WebGL.

Your Music

Add any audio file from your device. Upload a cover photo, set the artist name, and it lives in your library across page refreshes via IndexedDB.

No Server Needed

Groove runs entirely in the browser. Open the HTML file locally and it works. Everything — audio, covers, library — stays on your machine.
